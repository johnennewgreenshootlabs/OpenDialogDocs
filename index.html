Design Conversations

The OpenDialog conversation builder, with it's flexible scene-based approach, allows you to build complex conversations quickly and easily.

Build Actions

Our action framework lets you define actions within conversation flow and build the code to perform it.

Manage Content

We separate conversation flow from conversation content, making it easier for teams to collaborate and content to be managed.

Measure and Improve

We are working towards built-in analytics that will allow you to easily understand how your chatbot is performing.

Bring your NLP. Run on any platform.

OpenDialog integrates with any NLP tool and can be configured to run the same chatbot on multiple platforms.
